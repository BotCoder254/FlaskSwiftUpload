Installation

Requirements

  • Python 3.7 or higher
  • Flask 2.0 or higher
  • MongoDB (optional, for GridFS support)

Installation Methods

Using pip

pip install flask-swift-upload

This is the recommended method for most users.

Using Poetry

poetry add flask-swift-upload

From Source

git clone https://github.com/username/flask-swift-upload.git
cd flask-swift-upload
pip install -e .

Verify Installation

You can verify the installation by running Python and importing the package:

from flask_swift_upload import SwiftUpload

# Should print the version number
print(SwiftUpload.__version__)
